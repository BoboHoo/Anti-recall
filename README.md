## 本软件在 `v5.2.0` 后不再开源

最开始是因为本软件所需的权限很高, 为了证明没做坏事所以开源

但现在发现有人拿我的代码加了广告加了引流, 重新发布app, 在用百度搜索时还排到很前面, 非常寒心

同时因为开始商用, 固不再开源, 敬请谅解
